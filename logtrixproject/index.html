Built For Modern Transport Operations

Logtrix simplifies day-to-day logistics work for truck owners and fleet operators by centralizing trip logging, expense capture, and earnings visibility into one clean workflow. It is designed to be practical in the field, fast enough for high-volume use, and reliable enough to trust during real operations.

Key Features

Profile-Aware Automation

Select a registered truck ID and Logtrix automatically looks up and pre-fills the assigned driver's information instantly.

Dynamic Multi-Trip Entry

Effortlessly log complex days with dynamic rows for multiple trips, material descriptions, weights, and fuel expenses.

Instant Financial Calculations

Automatically calculates total rent, driver allowances (Batta), and final balances the second you finish a trip.

Offline-First Reliability

Data is stored immediately in a fast local database and syncs to the cloud when connectivity returns.

Expandable Dashboard

View the last 10 days chronologically, filter by route, and expand detailed cards to review historical trip expenses and earnings.

Built For Real Operators

Every screen is designed to reduce admin friction, improve accuracy, and help busy logistics teams focus on growth instead of paperwork.

Built With Modern Tech

Frontend Flutter and Material Design 3 for a fluid, polished mobile UI.
State Management Riverpod for fast, reactive updates and clean app architecture.
Database Isar NoSQL for lightweight, high-speed local storage offline.
Cloud Sync Firebase for backup, sync, and connected access when online.
